# Gigcaster 8 Firmware Update - Troubleshooting

## Common Issues

### Device Not Detected

**Problem:** Computer doesn't recognize Gigcaster 8 as GCS-8_INT drive

**Solutions:**
1. Ensure USB cable supports data transfer (not just power)
2. Try different USB port (USB 2.0 or 3.0)
3. Restart Gigcaster 8 in update mode:
   - Turn off device
   - Press and hold [MARK] button
   - Turn on power
   - Wait for "PREPARING..." then "CONNECTED"

### Update Fails

**Problem:** Update process stops or fails

**Solutions:**
1. Ensure firmware files are in root directory (no subfolders)
2. Verify all files were copied completely
3. Check USB connection is stable
4. Do not disconnect during "WRITING..." phase
5. Try different USB cable

### Files Won't Copy

**Problem:** Cannot copy files to GCS-8_INT drive

**Solutions:**
1. Check disk space on device
2. Ensure device is in update mode (not normal operation)
3. Try copying files one at a time
4. Check file permissions

## Recovery

If firmware update fails and device won't boot:

1. Contact Roland Support
2. Provide device serial number
3. Describe error messages displayed
4. May need service center repair
